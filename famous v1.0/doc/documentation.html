Introduction

Famous is Responsive One Page Music Band Template built with Bootstrap v4, HTML5 and CSS3. This template ideal for for music band, artists and musician.

Well Organized and valid Code.

This Documentation provide you a clear guideline for Famous to use and customize template. If you facing any problem, please don't forget to ask any question or contact with us for support. Thank You for Purchasing our Product.

Features

Famous comes with several features:

HTML5 & CSS3
Built with Bootstrap 4
LESS files included
Responsive design
One Page
Well documented
Google Fonts Used
Easy Customization

Folder Structure

Once you download the template, you can see a folder with several files and sub folders.

  • Famous
    • dist - Folder Main Template
      • css
        • bootstrap.min.css — Minified Bootstrap CSS files
        • styles.css — Compact version main CSS Files
        • styles.min.css — Minified version CSS files
      • Images — Here is a place to put your media files like logo, images, videoes, etc.
      • js
        • plugins
          • bootstrap.min.js — Bootstrap javascript files
          • popper.min.js — Popper javascript files
          • jquery.min.js — JQuery javascript files
        • scripts.js — Main javascript files
      • favicon.ico
      • index.html
      • detail-blog.html
    • doc - Folder Documentation
    • src - Folder Source

LESS, CSS Files & Structure

We are using LESS CSS pre-processor to develop this templates.

In folder src (source) you will found folder with name "less". This folder contain 3 file with extention .less. i.e. base.less, responsive.less, styles.less

base.less — Less css basic settings.

responsive.less — Less css responsive settings.

styles.less — The main css templates.



Result of process generating less file :

styles.css— compact version

styles.min.css— minified version



Every part of LESS File is seperated with Comments.

Javascript

Famous imports following plugins javascript files.

  1. jQuery
  2. Popper
  3. Bootstrap

and, Famous uses scripts.js for write main javascript file.

All code are beautifully written. You can customize easily. Just look the js code.

Example: main javascript structure — scripts.js

// Famous 1.0
$(document).ready(function() {

    $('.navbar-toggler').on('click', function(e) {
        e.preventDefault();
        $('.navbar').addClass('sticky');    
    });

   
    // Smoth Scroll
    $('a[href^="#"]').on('click', function(e) {
        e.preventDefault();

        var target = this.hash;
        var $target = $(target);

        $('html, body').stop().animate({
            'scrollTop': $target.offset().top - 79
        }, 900, 'swing');

        $('.navbar-collapse.collapse').removeClass("show");
    }); 

    //Sticky
    $(window).scroll(function() {
        if ($(window).scrollTop() > 0) {
            $('.navbar').addClass('sticky');
        } else {
            $('.navbar').removeClass('sticky');
        };
        if ($(window).scrollTop() > 50) {
            $('.scroll-to-top').addClass('affix');
        } else {
            $('.scroll-to-top').removeClass('affix');
        };

    });

    // Carousel
    var $item = $('.carousel-item'); 
    var $wHeight = $(window).height();
    $item.eq(0).addClass('active');
    $item.height($wHeight); 
    $item.addClass('full-screen');

    $('.carousel img').each(function() {
      var $src = $(this).attr('src');
      var $color = $(this).attr('data-color');
      $(this).parent().css({
        'background-image' : 'url(' + $src + ')',
        'background-color' : $color
      });
      $(this).remove();
    });

    $(window).on('resize', function (){
      $wHeight = $(window).height();
      $item.height($wHeight);
    });


   
});
